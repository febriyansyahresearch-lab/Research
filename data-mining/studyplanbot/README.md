# StudyPlanBot

StudyPlanBot is an academic chatbot designed to assist students with
study planning and academic consultation.  
This project was developed as part of a **Data Mining** coursework
for the **Masterâ€™s program in Information Technology (MTI)**.

## Objectives
- To design an academic chatbot for study planning and consultation.
- To apply data mining concepts in evaluating academic risk.
- To provide recommendations for course load (SKS) and thesis readiness.
- To demonstrate an interactive chatbot interface using Jupyter/Google Colab.
